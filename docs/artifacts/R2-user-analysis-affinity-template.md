## Final Problem Statements

Use this format:

> Mobile users need a way to [goal] because [evidence-backed problem], especially when [context].

Example:

> Mobile users need a faster way to find player ratings because long lists create repeated scrolling and memory load, especially when users do not remember the exact player name.
